# Using VaseMaker with PatternMaker

[PatternMaker](https://patternmaker.dcity.org) is a companion app that generates seamless tile patterns as SVG. Apply them as surface textures on your vases.

## Workflow

1. **Design your pattern** in [PatternMaker](https://patternmaker.dcity.org)
2. **Export SVG** — clean vector file, two colors
3. In **VaseMaker**, open the Textures section
4. Enable **SVG Pattern** texture
5. Click **Load SVG** (or **Paste SVG** to paste the SVG code directly)
6. Adjust **# Tiles Around** — how many times the pattern wraps the vase
7. Adjust **# Tiles Vert** — vertical repeat count
8. Set **Depth** — how deep the pattern cuts in mm

## Tips

- **High-contrast patterns work best** — pure black/white SVGs give crisp, clean relief
- **Grayscale SVGs** produce multi-depth effects — darker areas cut deeper. This can be beautiful for subtle textures
- For **lattice/cutout vases**, enable the **Cutout** toggle — dark SVG areas become holes through the wall
- Use **Smooth Zones** (base/rim %) to keep the top and bottom solid even with a cutout pattern
- PatternMaker's patterns are designed to tile seamlessly — they eliminate edge matching issues

## Recommended PatternMaker Patterns for Vases

| Pattern | VaseMaker Effect |
|---------|-----------------|
| Voronoi | Organic cell texture, looks like cracked clay |
| Truchet | Flowing arc patterns, intricate lattice |
| Hexagon Grid | Clean geometric grille, great for cutouts |
| Fractal Carpet | Bold geometric lattice with recursive holes |
| Reaction-Diffusion | Organic labyrinths, very natural-looking |
| Shapes Aligned | Regular geometric cutout patterns |
